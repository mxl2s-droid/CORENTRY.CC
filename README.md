# Corentry Developer Site

Static developer website for Corentry, an independent developer focused on video editing and playback apps.

## Files

- `index.html` - responsive developer homepage.
- `styles.css` - Apple-inspired visual system and responsive layout.
- `assets/corentry-logo.svg` - custom Corentry logo and favicon.
- `app-ads.txt` - reserved root document for authorized ad seller records.

Open `index.html` directly in a browser, or deploy the folder to any static hosting provider. Keep `app-ads.txt` at the domain root when deployed.
